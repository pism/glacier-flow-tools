# pypism

A set of random tools

## Installation

Get pypism source from GitHub:

    $ git clone git@github.com:pism/pypism.git
    $ cd pypism

Optionally create Conda environment named *pypism*:

    $ conda env create -f environment.yml
    $ conda activate pypism

Install pypism:

    $ pip install -e .
